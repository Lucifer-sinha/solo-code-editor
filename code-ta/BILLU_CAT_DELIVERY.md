```css

--magenta: #ff00ff;
--green: #00ff88;
```

### Change Code Content
```typescript
// Edit src/components/3D/MonacoPlayback.tsx
const codeLines = [
  '// Your custom code',
  'import React from "react";',
];
```

### Change Terminal Commands
```typescript
// Edit src/components/3D/TerminalPlayback.tsx

  '$ your-command',

];
```

